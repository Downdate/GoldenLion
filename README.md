## English

### Overview

**Frontend Learning** is an educational web application designed to help developers learn and practice frontend development concepts. This project demonstrates best practices in modern web development using vanilla JavaScript, HTML5, and CSS3.

### Features

✨ **Modern Components** - Reusable, well-structured components  
🎨 **Custom Styling** - Beautiful and responsive CSS  
⚡ **Fast Build** - Parcel bundler for optimized builds  
📱 **Responsive Design** - Mobile-first approach  
🚀 **Easy to Extend** - Clean architecture for adding new features  

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd GoldenLion
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

   The application will open in your browser at `http://localhost:1234`

### Available Scripts

```bash
# Start development server
npm start

# Build for production
npm build
```

### Project Structure

```
GoldenLion/
├── components/          # Reusable UI components
├── css/                 # Stylesheets
├── js/                  # JavaScript files
├── images/              # Image assets
├── index.html           # Main HTML file
├── package.json         # Project dependencies
└── .parcelrc            # Parcel configuration
```

### Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and animations
- **JavaScript (ES6+)** - Application logic
- **Parcel** - Module bundler
- **cssnano** - CSS optimization
